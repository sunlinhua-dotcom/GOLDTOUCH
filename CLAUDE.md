# Mojin Compute (摸金算力)

## 常用指令 (Commands)

- **启动开发服务器**: `npm run dev`
- **构建生产版本**: `npm run build`
- **运行生产版本**: `npm run start`
- **代码检查**: `npm run lint`
- **数据库管理**: `npx prisma studio`

## 技术栈 (Tech Stack)

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: CSS Modules (Vanilla CSS) - Keep it Premium & Minimalist
- **Database**: SQLite + Prisma ORM
- **AI**: Google Gemini API

## 目录结构

- `src/app`: 页面路由 (Pages)
- `src/components`: UI 组件 (Components)
- `src/lib`: 核心逻辑 (Prisma, Gemini Client)
- `prisma`: 数据库模型 (Schema)
